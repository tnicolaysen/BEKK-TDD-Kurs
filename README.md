### Kom igang

* Import ved klikk. 
* Bruk standardvalg gjennom guiden.
* `⌘+N` / `CTRL+N` etterfulgt av å skrive inn BGT (BowlingGameTest). Enter. 
* Deretter `SHIFT+F10` Enter for å kjøre testen.
